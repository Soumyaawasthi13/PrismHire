
# PrismHire
### AI-Powered Candidate Discovery and Ranking Platform

PrismHire is an AI-powered recruitment intelligence platform that helps recruiters discover and rank candidates using semantic search, candidate DNA profiling, and multi-agent scoring.

## Features

- Semantic Candidate Search using Sentence Transformers
- Candidate DNA Profiling
- Multi-Agent Ranking Engine
- Explainable AI Recommendations
- Interactive Gradio Dashboard

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Sentence Transformers
- FAISS
- Gradio

## Candidate Ranking Formula

Final Score =
0.30 × Semantic Score
+ 0.25 × Technical Score
+ 0.15 × Career Score
+ 0.10 × Market Score
+ 0.10 × Recruitability Score
+ 0.05 × Experience Score
+ 0.05 × Role Score
